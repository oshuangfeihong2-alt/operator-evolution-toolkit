## L1 · 权力基础——没权力认知，一切都是空谈

### 1.1 七大权力特质自评（《权力》）

逐条打分（1-5），你缺哪条练哪条：

```
□ 雄心：愿意付出超常努力、做出牺牲的驱动力
□ 精力：能感染他人、保证长时间高强度工作
□ 专注：聚焦于特定行业/关键任务，不分散
□ 自知之明：结构化自我反思，通过复盘持续修正
□ 信心：表现得有能力、有见识以扩大影响力
□ 换位思考：站在他人角度理解动机和需求
□ 化解冲突：在高压和冲突环境中有效运作
```

### 1.2 硬球14条博弈法则（精简速查）

| # | 法则 | 一句话操作 |
|---|---|---|
| 1 | 关键不在认识谁，而在想认识谁 | 主动出击锁定目标人物 |
| 2 | 一切政治都是乡土的 | 抓住和对方个人关系最大的事 |
| 3 | 索取比给予还好 | 让他在你身上下赌注，投入越多越忠诚 |
| 4 | 与带你来的人共舞 | 忠诚是金科玉律；慎选向谁效忠 |
| 5 | 让你的敌人站在你的前面 | 把对手放在你能监督的位置上 |
| 6 | 不要疯狂，不要扯平，向前推进 | 不浪费生命在复仇上 |
| 7 | 对任何飞来之箭，一一反击 | 24小时内不回应，谎言变真理 |
| 8 | 说话是为了更好地沉默 | 多问少说；沉默是终极权力武器 |
| 9 | 永远对原则问题表示赞同 | 口头让步换实质利益 |
| 10 | 举灯照亮你的问题 | 自己做自己的告密者，主动暴露缺点 |
| 11 | 让对手高飞，让对手惨败 | 把对手逼到不舒服的位置 |
| 12 | 让媒体为你免费宣传 | 在记者心中"播下种子" |
| 13 | 把一个人送上巅峰 | 创造英雄，他的光芒会反射到你 |
| 14 | 说话时直视镜头 | 在公共场合永远表现强大 |

### 1.3 权力建设四个核心动作

```
1. 选择离权力近的位置
   → 判断标准：部门薪酬水平 + 办公室距高管的物理距离 + 是否进入委员会
   
2. 进入创造价值的部门（P&L部门而非成本中心）
   → 能直接为公司赚钱/省钱的岗位，话语权天然更大

3. 成为人际关系网络的结构性空洞
   → 做那个"连接两个互相不认识的群体"的人
   → 弱联系比强联系更有用——认识大量各行各业的人

4. 创造独一无二的资源
   → 第一个发起会议/论坛/项目的人，获得"先发垄断权"
```

## L2 · 谈判武器——四本经典融合为一条完整战线

### 2.1 原则谈判四要素（《谈判力》BATNA源头）

```
要素1 · 把人和事分开
  "对人温和，对事强硬。"
  操作：把关系利益与实质利益分开处理
  从认知、情绪、交流三个维度处理人际问题

要素2 · 着眼于利益，而非立场
  "立场是你已做的决定，利益是导致你做决定的原因。"
  操作：每个立场背后问"为什么？"→挖出真正利益
  同一利益可以有多种满足方式

要素3 · 为共同利益创造选择方案
  操作：把"创造方案"和"做决定"分成两个独立步骤
  先头脑风暴（不判断），再筛选决策

要素4 · 坚持使用客观标准
  操作：用"公平标准"（市场价/专家意见/惯例/法律）取代"谁说了算"
  公平程序："一人切，另一人选"
```

### 2.2 BATNA分析（谈判桌前必做的功课）

```
BATNA = Best Alternative To a Negotiated Agreement
       = 如果这次谈不成，你最好的替代方案是什么？

BATNA三步分析：
  1. 列出如果谈判失败，你所有的替代选项
  2. 选出最好的那个替代选项
  3. 把对方的BATNA也分析一遍

谈判力公式：谁的BATNA更强，谁在谈判桌上更有力。

致命错误（没有之一）：
  ❌ 不知道自己的BATNA就去谈判
  ❌ 高估自己的BATNA
  ❌ 低估对方的BATNA
```

### 2.3 优势谈判三阶段战术速查（《优势谈判》核心）

**开局阶段**：
```
● 永远让对方先出价（除非你掌握了绝对的信息优势）
● 对方出价后做"退缩反应"（不是演戏，是制造锚定效应偏移）
● 绝不接受第一次报价（给对方"赢的感觉"）
● 用"更高权威法"："我个人可以同意，但我需要跟合伙人确认一下。"
```

**中场阶段**：
```
● 永远不做单方面让步（每次让步换一个对等让步）
● 让步幅度逐次递减（50%→25%→12.5%——对方知道你接近底线了）
● 用"交换"不是"妥协"："如果你能在这一项上让步，我可以在另一项上给你补回来。"
● 用沉默制造压力——谁先开口谁先让步
```

**终局阶段**：
```
● 白脸/黑脸策略识别和应对
● "蚕食法"：在最后阶段提一个很小的额外要求
● "定金法"：让对手在这笔交易上投入时间和精力，投入越多越难退出
● 成交后不要得意忘形——第一时间书面确认
```

### 2.4 应对肮脏谈判策略速查表

| 对方使用 | 你的应对 |
|---|---|
| 故意撒谎 | 用客观标准验证："这个数字我回去核实一下。" |
| 人身攻击 | 识别但不反应："我们来谈谈问题本身。" |
| 威胁 | 沉默→识别对方的BATNA→评估威胁的可信度 |
| 黑白脸 | 直接指出："你们这是在演黑白脸吧？" |
| 故意冷场 | 安静地等——沉默是你的武器 |
| 最后通牒 | "我理解这是你的立场。如果这是我们能找到的最好方案，那很遗憾我们可能达不成一致。" |
| 不断升级要求 | 每新加一个要求，要求对等交换 |

## L3 · 中国式博弈——在东方文化里玩权力游戏

### 3.1 圆通 vs 圆滑（《圆通的人际关系》核心）

| | 圆通 | 圆滑 |
|---|---|---|
| 动机 | 尊重对方、维护关系 | 利用对方、达到目的 |
| 表现 | 话不说满、留有余地 | 口是心非、见人说人话 |
| 结果 | 对方舒服、长期信任 | 短期见效、长期失信 |
| 本质 | 有原则的灵活 | 无原则的迎合 |

**圆通操作八法**：
1. 话不说完——"我回去琢磨琢磨"（留余地）
2. 事不做绝——给对手留退路（日后好相见）
3. 面要给足——公开表扬、私下批评
4. 理要占住——无论结果如何，道理上不能被动
5. 气要忍得住——不在情绪最高点做决定
6. 时机会选——有些话说早了没用，说晚了也没用
7. 人要分得清——谁是自己人、谁是中间人、谁是对手
8. 账要算得明——人情账和经济账分别记账

### 3.2 六角沟通法（《有言以对》核心）

```
史欣悦（顶尖律所合伙人）的六维沟通框架：

① 逻辑力   —— 想清楚再开口。论点→论据→逻辑链
② 表达力   —— 把复杂的事说简单。一次只说一个重点
③ 共情力   —— 站在对方鞋里想。先理解他的难处再讲你的道理
④ 应变力   —— 对方出招你接得住。不正面冲突，借力打力
⑤ 控局力   —— 把握对话节奏。该快则快，该停则停
⑥ 影响力   —— 让对方觉得按你说的做是他的主意
```

### 3.3 面子管理三原则（中国式博弈必修课）

```
1. 给面子
   → 公开场合给足面子（称呼、座次、发言顺序）
   → 即使博弈失败也让人家有台阶下

2. 留面子  
   → 批评时先肯定后提建议
   → 不让他人在自己下属面前丢脸

3. 争面子
   → 有成绩时适当展示（不过分谦虚也不夸耀）
   → 别人帮你挣了面子，要公开感谢
```

## L4 · 股权与合伙——钱的游戏才是终极游戏

### 4.1 控制权八种武器（《股权战争》17案例提炼）

```
武器1 · 股权比例  —— 67%（绝对控制）/ 51%（相对控制）/ 34%（一票否决）
武器2 · 一致行动人  —— 多个股东签署一致行动协议，合并投票权
武器3 · 双层股权结构 —— A股1票/B股10票（AB股），同股不同权
武器4 · 董事会过半   —— 控制董事会席位比控制股权更重要
武器5 · 一票否决权   —— 特定事项上否决权。写进章程
武器6 · 投票权委托   —— 让其他股东把投票权委托给你
武器7 · 合伙人制度   —— 控制合伙人的提名权和进入门槛
武器8 · 章程设计     —— 在章程里预设对自己有利的游戏规则
```

**最经典的教训**：股权比多少不如控制权握在谁手里。51%的股权也可能失去公司——关键看董事会和章程。

### 4.2 合伙人翻车五大典型模式

```
1. 股权平分 → 50:50是世界上最糟糕的股权结构（谁说了都不算）
2. 没有退出机制 → 有人想走，但没人知道怎么走
3. 贡献与股份脱节 → 出力的人拿得少，拿得多的人不干活
4. 夫妻/亲戚合伙 → 感情破裂牵连公司（"土豆条款"——离婚影响上市）
5. 口头约定代替书面协议 → 翻脸时各说各的
```

### 4.3 合伙协议关键条款清单（《合伙人原则》）

```
□ 角色与责任：谁负责什么，决策权边界
□ 出资与股份：投入什么（钱/技术/资源），占多少
□ 利润分配：怎么分钱，什么时候分
□ 决策机制：哪些事需全票通过，哪些事多数即可
□ 退出机制：怎么退出，股份怎么估值，谁有优先购买权
□ 僵局解决：双方争执不下时的解决机制（第三方仲裁/德州枪手条款）
□ 竞业禁止：退出后多久不能从事同类业务
□ 知识产权：公司成果归谁，个人带进来的IP归谁
□ 离婚/去世条款：配偶不因离婚或继承自动成为合伙人
```

## L5 · 社交战场——权力不是在会议室里角逐的，是在饭桌上

### 5.1 饭局政治操作全流程（复用第二批《饭局人脉学》）

**饭局的本质**：不是吃饭，是信息交换+关系兑换+信任测试+联盟建立。

| 阶段 | 博弈操作 | 禁忌 |
|---|---|---|
| **组局** | 邀请谁=传递什么信号；不邀请谁=传递什么信号 | 最忌讳请了A不请B，B和A认识 |
| **座次** | 主人坐哪里、主宾坐哪里、谁挨着谁是精心安排 | 坐错位置比不来还糟糕 |
| **话题** | 第一阶段闲聊→第二阶段试探→第三阶段切入正事 | 一上来就谈正事（"你是来吃饭的还是来谈事的"） |
| **敬酒** | 顺序=权力顺序。杯沿低于对方=态度 | 对方不喝你硬劝 |
| **后续** | 第二天发信息而非当天："昨天聊到的那个事，我整理一下。" | 饭局结束人就消失 |

### 5.2 向上社交34法精简精华（《向上社交》）

**核心原则**：优秀的人愿意帮你的前提是——你不是来索取的，你是来提供某种价值的。

```
建立连接的五个入口：
1. 真诚的好奇心（不是奉承，是真的对他的领域有兴趣）
2. 提供具体价值（一个他可能不知道的信息/资源/视角）
3. 共同点破冰（共同的母校/行业/经历/观点）
4. 提出一个好问题（好问题比好答案更稀缺）
5. 介绍他认识另一个有价值的人（三角连接最受欢迎）
```

**高位者社交5条铁律**：
1. 不浪费他的时间——提前准备好要问什么
2. 不做伸手党——先给予再请求
3. 让他感觉被尊重而非被利用——真诚大于技巧
4. 保持联系但不骚扰——偶尔更新你的进展
5. 永远感谢——没有人有义务帮你

## ⚠️ 权力博弈七大反模式

| # | 反模式 | 为什么错 | 正确做法 |
|---|---|---|---|
| 1 | **相信世界是公平的** | "只要把工作做好自然有人看到"——不会 | 主动建立影响力，主动让对的人看到 |
| 2 | **把权力当脏东西** | "玩政治的人都不干净"——你不玩，玩你的人不会停 | 权力是中性的，看你怎么用 |
| 3 | **没有BATNA就去谈判** | 你不知道谈崩了怎么办 | 谈之前一定想清楚BATNA |
| 4 | **把暴露弱点当诚实** | "这是我的底价"——那你就输了 | 诚实≠交出底牌 |
| 5 | **赢了谈判输了关系** | 这次你赢了，下次他绕着你走 | 给对手留面子、留台阶 |
| 6 | **在敌人看不到的地方对抗他** | 你不知道他在干什么 | 法则5：把敌人放在看得到的地方 |
| 7 | **合伙时没有书面协议** | "我们关系这么好不需要"——翻脸时就是灾难 | 先小人后君子，把丑话说在前面 |
