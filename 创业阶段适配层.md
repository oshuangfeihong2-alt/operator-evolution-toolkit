# 🎯 创业阶段适配层 (Stage-Adaptive Decision Layer)

> **v2.0 核心升级 · 解决"大企业方案误用到小企业"的根本问题**
>
> 一切建议发放之前，必须先过三道判断。从此Skill不会对你一个单打独斗的人推荐"事业部制"。

---

## 阶段诊断四象限

每次你描述处境，Skill先判断你在哪个阶段：

```
阶段0 · 验证期（1-3人，无收入）
  ├─ 核心任务：找到Product-Market Fit（产品-市场匹配）
  ├─ 不适合读：OKR、绩效体系、企业文化手册、事业部制
  ├─ 适合读：精益画布、客户访谈(Mom Test)、MVP、Running Lean
  └─ 一句话：还没验证有人愿意付钱之前，其他都是虚的

阶段1 · 生存期（3-10人，有初步收入但不稳定）
  ├─ 核心任务：跑通现金流、找到可重复的销售模式
  ├─ 不适合读：规模化组织架构、全面预算管理
  ├─ 适合读：跨越鸿沟、增长黑客、格鲁夫管理杠杆率
  └─ 一句话：活下来是唯一目标，所有"规范"都可以往后放

阶段2 · 增长期（10-50人，有稳定模式，在扩张）
  ├─ 核心任务：建团队、建流程、可复制地增长
  ├─ 不适合读：单打独斗的生存技巧（已过时）
  ├─ 适合读：德鲁克"用人所长"、格鲁夫1-on-1体系、绩效评估
  └─ 一句话：从"你自己能干"到"通过别人能干"的转型

阶段3 · 规模化期（50人+，模式成熟）
  ├─ 核心任务：体系、文化、第二曲线
  ├─ 适合读：德鲁克全面管理体系、从优秀到卓越、基业长青
  └─ 一句话：你不是在管公司，你是在建系统
```

## 德鲁克争议的精确回应

**问**：德鲁克适不适合小企业？
**答**：《卓有成效的管理者》全书适用任何阶段——"时间管理、聚焦贡献、要事优先、有效决策"跟公司大小无关。
《管理的实践》《成果管理》中的事业部制、战略规划、管理层级——阶段2+适用，阶段0-1 看了也没用。

**Skill不会因为"德鲁克是大企业专家"就一刀切不用他，也不会因为"德鲁克是经典"就全盘推给你。每一条建议都根据你的阶段过滤。**
