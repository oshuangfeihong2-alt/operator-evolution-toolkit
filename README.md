
## 关于"蒸馏"

**蒸馏和上传到知识库有什么区别？**

- 上传知识库（RAG）：你问什么它搜原文一段给你。你问"怎么写好开场"，它给一段书中原文，你还得自己读、自己消化。
- 蒸馏：把整本书的方法论拆开重组——框架、原则、技法、反模式全部结构化。你问"怎么写好开场"，它直接用书里的框架给你出完整可执行步骤。

**什么书能蒸馏？**

- ✅ 能做：框架清晰的方法论书——商业、思维、写作、沟通、自我管理类
- ❌ 不能做：纯小说、纯叙事、纯理论学术书、数据字典类参考书
